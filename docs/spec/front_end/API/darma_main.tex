\subsection{\texttt{darma\_main}}

\paragraph{Summary}\mbox{}\\
\codetarget{darma_main}
is the entry point \gls{DARMA} uses to launch the user's code.

\paragraph{Syntax}\mbox{}\\
\begin{CppCode}
int darma_main(int argc, char** argv)
\end{CppCode}

\paragraph{Details}\mbox{}\\
The signature of \codelink{darma_main} mimics the signature of |int main()|,
which \CC{} uses as the entry point to user code.

\paragraph{Code Snippet}\mbox{}\\ 
\begin{figure}[!h]
\begin{CppCodeNumb}
#include <darma.h>
int darma_main(int argc, char** argv)
{
  return 0;
}
\end{CppCodeNumb}

  \caption{Basic usage of \protect\codelink{darma_main}.}
\end{figure}
